# Учёт продаж церковной лавки

Одностраничное веб-приложение для учёта продаж товаров церковной лавки. Интерфейс работает локально в браузере: данные сохраняются в `localStorage`, поэтому можно вести смену без сервера и базы данных.

## Возможности
- Добавление категорий и товаров с ценой и выбором по умолчанию (QR или наличные).
- Ускоренный приём оплаты: выбирайте товар, способ оплаты и регистрируйте продажу одним нажатием.
- Отчёты по товарам и категориям, итоговые суммы по QR и наличным платежам.
- Выгрузка отчёта в CSV для сохранения или передачи бухгалтерии.
- Экспорт и импорт каталога категорий/товаров в JSON для переноса настроек между устройствами.
- Сброс смены с очисткой всех продаж.

## Запуск
1. Склонируйте репозиторий или скачайте файлы.
2. Откройте `index.html` в современном браузере (Chrome, Firefox, Edge или Safari). Дополнительный сервер не требуется.

## Работа с данными
- Каталог (категории и товары) и продажи сохраняются в `localStorage` браузера. 
- Для резервного копирования каталога используйте кнопки «Экспорт каталога» и «Импорт каталога» на вкладке «Настройки».
- Для очистки статистики текущей смены нажмите «Сбросить смену». Каталог при этом не изменится.

## Формирование отчёта
На вкладке «Отчёт» доступна сводка:
- таблица по каждому товару: количество продаж и суммы по QR и наличным;
- блок «По категориям» с количеством и суммой по каждой категории;
- итоговые суммы по способам оплаты и общая выручка.

Кнопка «Скачать CSV» сохраняет эти данные в файл с меткой времени, чтобы его можно было открыть в Excel или передать в бухгалтерию.
